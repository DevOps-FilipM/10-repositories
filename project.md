# Project Idea: Infrastructure Monitoring System

## Description
A web application for real-time monitoring of servers and services.

## Features
- Dashboard with CPU, RAM, disk metrics
- Email/Slack alerts when thresholds are exceeded
- Metrics history with charts
- Multi-server management

## Tech Stack
- Backend: Python (Flask)
- Frontend: HTML/CSS/JavaScript
- Database: PostgreSQL
- Monitoring: Prometheus + Grafana

## Goal
Learning how to integrate DevOps tools in a single project.
